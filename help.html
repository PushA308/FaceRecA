How to get started with FaceRecA ?






This section contains a wealth of information, related to FaceRecA and its feature. If you cannot find an answer to your question, make sure to contact us.


General questions :

Is account required?

Account for login at FaceRecA is required if you are faculty of the institute.This ensure the valid authorization to monitor the class attendance.

Can anyone create new user from registration?

Only System Administrator of the department is allowed to create the account for the faculties as per their courses. This registration process is not accessible for everyone.

Teacher section

How to get started?

All departmental teacher will get the username and password from the system administrator,Which help them to login to FaceRecA

What to do after Logged in?

The steps involved in this process are really simple. All you need to do is:

  • Slect the class and respective course-code of the perticular subject.

  • Upload an image using browse option for generating the attendance.

  • Attendance will be generated and updated in the attendance-table.

  • In the Attendance-table teacher also can easily modify the status of the student as 'present' or 'absent' if it is required.

  • Uploaded Images will be stored in the folder.
